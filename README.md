# react-tailwind-template

- react
- tailwind



## how to install


```bash
    git clone https://github.com/alipnf/react-tailwind-template.git
```
```bash
    rm -rf react-tailwind-template/.git/
```
```bash
    mv react-tailwind-template/ project-name
```
```bash
    cd project-name
```
```bash
    pnpm install
```
```bash
    pnpm run dev
```
